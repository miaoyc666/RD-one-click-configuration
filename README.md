# RD-one-click-configuration
R&D environment one click configuration

### Usage
```bash
./install.sh
```

### OS support
- Centos 7
- Ubuntu 22.04 LTS

### How to do
- install ccat
- mirrors source config: yum or apt
- install tmux
- copy .bashrc
- copy .vimrc
- copy .gitconfig

### Software List
- ccat
- vim
- tmux
- htop
